我，一个前端攻城师。
入行时间：2013年6月~至今

起初学的是java，后面因为个人爱好和工作需求便走上了前端开发这条不归路！！
而对于我自己也很喜欢这样一条路。

我的前端发家之路是从ExtJs开发开始的，因为太依赖于框架，而原生写法用的比较少，所以导致了我在一段时间内对Javascript基础并不是很牢靠~现在已经恶补回来。这一定请放心。

而对于HTML和CSS方面，我自认为自己属于形象记忆思维，所以对于这方面参透性还可以，也因此这一块也不算什么软肋[但也不是大神哦~]

当前技能点：

  1. HTML + Javascript + CSS
  2. jQuery
  3. Extjs 4.x [现在工作中一直在用(以用一年)]
  4. AngularJs [自学的并且保证可以上手项目了]
  5. PHP [用thinkPHP做过一个小项目]
  6. CSS预编译工具 [用过Less和Sass 但没有系统性的实战过]
  7. 版本管理工具(SVN、Git) [一直在用，从未停止]
  8. 前端架构工具(Grunt、Bower、Yoeman) [使用起来基本没问题]

给自己的定位和总结：少年无知，求学路上的我
